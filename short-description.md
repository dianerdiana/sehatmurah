**Forum Discussion API**

**_Dicoding Course Project_**

Developed a production-grade RESTful API for a forum discussion platform, applying Clean Architecture principles to separate domain logic, use cases, and infrastructure concerns, with automated quality gates via CI/CD and Git hooks.

- Tech Stack: TypeScript, Node.js (ESM), Express.js, PostgreSQL, JWT, bcrypt, Vitest, Husky, lint-staged, GitHub Actions
- Enforced strict layered architecture (Domains → Applications → Infrastructures → Interfaces) with dependency injection via instances-container; implemented JWT-based authentication with access & refresh token rotation; and built a full CI pipeline with type-checking, linting, testing, and auto-deploy on every push to main.

**Forum Discussion App**

**_Dicoding Course Project_**

Built a production-ready forum discussion web application with authentication, thread management, voting, and real-time leaderboards, applying a clean layered front-end architecture with strict separation between presentation, state, data, and validation concerns.

- Tech Stack: React 19, TypeScript, Vite, Redux Toolkit, React Router v7, Axios, TailwindCSS v4, Radix UI, ShadCN, React Hook Form, Zod, Vitest, Testing Library, Cypress, Storybook, ESLint, Prettier
- Structured global state with Redux Toolkit and async thunks for all side effects; enforced schema-based form validation using Zod; implemented token-based authentication with protected and public routes; and maintained code quality through unit, integration, and E2E testing alongside component-driven development with Storybook.

**Tooang - Digital Menu & Business Management**

**_University Capstone Project_**

Built a multi-tenant digital menu platform that allows restaurants to present digital menus and manage business data through an admin dashboard, aimed at helping MSMEs adopt digital operations efficiently.

- Tech Stack: Astro, React (Vite), NestJS, PostgreSQL (TypeORM), Google Cloud Platform (GCP)
- Reduced reliance on physical menu printing, delivered a ready-to-deploy solution for local restaurants, and demonstrated scalable full-stack architecture with clear separation between marketing, dashboard, and API layers.

**Book a Doctor App**

**_Capstone Project_**

Built a healthcare booking platform that connects patients, doctors, and administrators, enabling doctor discovery, appointment scheduling, secure document uploads, and centralized management of medical services.

- Tech Stack: React 19, TypeScript, Vite, TanStack Router, TanStack Query, Axios, TailwindCSS v4, Express.js, MongoDB, Mongoose, Zod, JWT, Multer
- Streamlined appointment workflows with patient and doctor dashboards, role-based access control, doctor approval and schedule management, and admin tools for managing users, specialists, appointments, and reviews.
