## Pro-Tips for Recording the Demo Video

1. **Resolution & Settings:** Record in **1085p (Full HD)** at **60fps** if possible. Keep your IDE font size slightly larger so code snippets are easily readable on smaller screens.
2. **Audio Quality:** Use a decent dedicated microphone. Record in a quiet room with minimal background echo.
3. **Pacing:** Speak clearly and at a moderate pace. Pause for a second after clicking a page or submitting a form to let the audience absorb the visual change.
4. **Tooltips & Highlighting:** Use a screen cursor highlight tool or zoom in slightly during post-production when explaining the backend folder structure or CASL abilities to draw the viewer's focus.
5. **Data Preparation:** Pre-seed a few doctors and reviews so the UI looks lively and premium. You can run the seeder script beforehand.
